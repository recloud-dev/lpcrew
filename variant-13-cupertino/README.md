# Variant 13 — CUPERTINO

Минимализм в духе [apple.com](https://www.apple.com). Сайт команды подан как продуктовая страница Apple: максимум воздуха, центрированный контент, крупная semibold-типографика, продуктовые секции и bento-сетка.

## Идея
Премиальный, тихий, выверенный. Контент «продаётся» через типографику, фото и пустоту — без украшений. Резкий контраст к ярким/тёмным вариантам.

## Дизайн-язык
- **Палитра** — белый + `#f5f5f7` (чередование секций), чёрные продуктовые секции, ink `#1d1d1f`, серый `#86868b`, синий акцент `#0071e3` (ссылки, CTA).
- **Типографика** — системный SF-стек (`-apple-system / SF Pro`), semibold-заголовки, тайтовый отрицательный трекинг, центрирование. Огромный hero-заголовок.
- **Форма** — крупные скругления (18–40px), pill-кнопки, синие текст-ссылки с шевроном `›`.

## Сигнатуры Apple, перенесённые на LP Crew
| apple.com | CUPERTINO (LP Crew) |
|---|---|
| Frosted blur navbar, тонкая, центр-линки | Стеклянная nav-бар (`saturate blur`), сплошной бордер при скролле |
| Hero: огромный заголовок + 2 синие ссылки + фото | «Плыви дальше.» + ссылки + кадр со scale-on-scroll |
| Продуктовые секции (light / dark) | «Открытая вода» (чёрная) · «Бассейн» (серая) |
| Bento-сетка плиток (свет/тьма/фото) | Возможности: тренировки, online, видеоразбор, сборы |
| Крупные центрированные цифры | Статистика команды (count-up) |
| Apple-футер: мелкие серые ссылки, дисклеймер | Многоколоночный футер с примечанием и юр-ссылками |

## Анимации и интеракции
- **Scroll-reveal** — fade-up (`opacity + translateY`) с фирменной кривой Apple, плюс stagger по плиткам/карточкам.
- **Scale-on-scroll** — кадры в hero и продуктовых секциях растут `0.92 → 1.0` по мере входа в экран (rAF), как продуктовый reveal Apple.
- **Frosted nav** — `backdrop-filter: saturate blur`, появление бордера при скролле.
- **Count-up** статистики.
- **Hover** — лёгкое «нажатие» плиток/карточек (`scale .985`), zoom фото в новостях.
- **Mobile** — бургер + полноэкранный blur-drawer.
- Уважает `prefers-reduced-motion` (отключает reveal, scale, анимации).
