# LP Crew — прототип нового дизайна

Статический HTML/CSS/JS-прототип редизайна сайта [lpcrew.ru](https://lpcrew.ru/).
Без сборки и зависимостей — открывается напрямую в браузере.

## Запуск
```bash
open index.html          # macOS
# или локальный сервер:
python3 -m http.server 8000   # → http://localhost:8000
```

## Структура
```
variant-1-aqua/
├── index.html        # главная (one-page)
├── news.html         # список новостей
├── article.html      # страница новости (?id=…) с каруселью фото
├── css/style.css     # дизайн-система + адаптив + стили новостей/карусели
└── js/
    ├── main.js       # бургер-меню, счётчики, reveal, демо-форма
    ├── news-data.js  # ЕДИНЫЙ источник новостей (массив статей + галереи)
    ├── news-list.js  # рендер списка на news.html
    └── article.js    # рендер статьи по ?id= + логика карусели
```

## Новости отдельными страницами
- Карточка новости (на главной и в `news.html`) ведёт на `article.html?id=<slug>`.
- Контент статей — в `js/news-data.js` (заголовок, дата, категория, обложка, лид, абзацы, массив фото). Добавить новость = добавить объект в массив; страницы перерисуются сами.
- **Карусель фото** на странице новости: стрелки, точки, счётчик, свайп на тач-устройствах, стрелки клавиатуры. Если фото одно — управление прячется.
- Перенос на Tilda: список — нативный блок-лента/каталог, страница новости — шаблон записи (Tilda «Блог»/«Магазин»-карточка) либо Zero Block + галерея-слайдер T184.

## Что внутри
- **Дизайн-система:** палитра воды (deep `#0a4d8c` → blue `#0094c6` → teal `#00b8a9`) + тёплый коралл `#ff6b5b` для CTA; шрифты Manrope (заголовки) + Inter (текст); 8-pt сетка отступов, единый radius/тени.
- **Блоки:** Hero с анимацией волн → статистика-счётчики → о команде → 4 услуги → тренеры → галерея/география → отзывы → новости → форма заявки с мессенджерами → подвал.
- **Интерактив:** count-up статистики при прокрутке, reveal-анимации, мобильное меню, демо-форма.
- **Адаптив:** mobile-first, брейкпоинты 900/560px. Уважает `prefers-reduced-motion`.

## Заметки по контенту
- Фото — **реальные**, скачаны с lpcrew.ru (лежат в `../img/`). Для боя пережать в WebP и привести к единому кадрированию.
- Блок «Тренеры» сейчас показывает командные фото (нельзя достоверно сопоставить лица с именами Камилия/Лера/Лилия) — заменить на портреты с подписями.
- Отзывы — примеры-рыба, заменить на реальные.

> Это **вариант 1 из 3**. См. также `../variant-2-noir/` (тёмный кинематографичный) и `../variant-3-splash/` (яркий энергичный). Витрина — `../index.html`.

## Перенос на Tilda
Весь макет реализуется через **Zero Block** + блок **T123 (HTML/CSS)**:
1. Палитру и шрифты задать в настройках проекта Tilda.
2. Hero/статистику/карточки собрать в Zero Block по этому макету.
3. CSS-анимации и счётчики — вставить из `css/style.css` и `js/main.js` в HTML-блок.
4. Форму — нативную Tilda-форму подключить к Telegram-боту/CRM.

Подробный анализ и обоснование — в `../ANALYSIS.md`.
